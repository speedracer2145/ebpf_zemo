00 before you paste this
what this project is

eBPF-Embed is a research tool that takes eBPF bytecode (.o files), extracts Function-level Control Flow Graphs (FCFGs), runs them through two parallel streams — a fine-tuned CLAP-ASM structural encoder and an LLM semantic summary generator — then fuses both into a single enriched embedding vector. Downstream uses: NL summary generation, semantic similarity detection, and as a drop-in embedding module for Yaksha-Prashna.

the three papers Claude needs to know
  • Bin2Summary (ACM 2024) — strips binary functions, uses CFG random walk + seq2seq for NL summaries. We replace the random walk embedding module.
  • CLAP (ISSTA 2024, arxiv 2402.16928) — Contrastive Language-Assembly Pre-training. Aligns assembly encoder with text encoder via NL supervision. We fine-tune clap-asm on eBPF-specific tokens.
  • Yaksha-Prashna (IIT Hyderabad, Feb 2026) — eBPF bytecode behavioral analysis system. Uses dataflow + CFG analysis + Prolog-like DSL. We plug in as its embedding module.
the core research insight

Structural encoders (CLAP) fail when CFG topology is similar but semantics differ. LLM summaries hallucinate on low-level register / map access details. Fusion fixes both: summary fills semantic gaps, structure anchors against hallucination. Nobody has done this for eBPF bytecode specifically. That's the contribution.

Their encoder was built for x86/ARM — we adapt it to eBPF. - Key eBPF tokens to inject: MAP_LOOKUP, MAP_UPDATE, XDP_DROP, XDP_PASS, XDP_REDIRECT, TC_ACT_OK, HELPER:bpf_redirect, HELPER:bpf_skb_store_bytes, BPF_REG_CTX, packet field reads/writes (skb->protocol, xdp_md->data) PAPER 3 — Yaksha-Prashna (IIT Hyderabad, Feb 2026, ResearchGate preprint) - eBPF behavioral analysis: queries bytecode conformance to spec - Uses dataflow analysis guided by CFG to extract: packet field ops, map accesses, helper calls, XDP actions - Has a Prolog-like DSL for expressing assertions and retrieval queries - OUR ROLE: We provide richer embeddings for their similarity detection. This grounds the LLM summary with structural precision.
